# Project Title
Knex, a React and Redux based knowledge exchange plattform.

## Installation
- clone the repository
- load dependencies via:

```
npm install
```
inside the folder 'web'

- start the application with:
```
npm run
```
- visit
```
http://localhost:8080
```
inside your browser to see the application.
