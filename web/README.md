# Knex
Knex, a React and Redux based knowledge exchange plattform.

## Node & npm Versions
- Node 7.10
- npm 4.6

## Installation
- clone the repository
- load dependencies via:

```
npm install
```
inside the folder 'web'


- start the application with:
```
npm start
```

- visit
```
http://localhost:8080
```
in your favourite browser to see the application.
